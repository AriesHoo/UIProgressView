#UIProgressView
--------------------------
####一、简介：
一个类似新浪微博、微信样式等待Loading效果
**1.1 Gradle集成**

```
allprojects {
    repositories {
        ...
        maven { url "https://jitpack.io" }
    }
}
```

```
dependencies {
     compile 'com.github.AriesHoo:UIProgressView:1.0.0'
}
```
